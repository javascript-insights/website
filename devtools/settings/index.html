To access Chrome settings, copy the URL below and paste it into a new tab:

chrome://settings/

What you can do with the Chrome settings page:

  • Manage your search engine preferences
  • Customize the appearance of your browser
  • Control privacy and security settings
  • Manage passwords and autofill data
  • Set up and manage browser extensions
  • Configure site settings and permissions
  • Manage your Google account and sync settings
  • Clear browsing data
  • Set up startup pages and behavior
  • Adjust download settings
  • Configure accessibility features
  • Manage languages and input settings
  • Control system settings like hardware acceleration
