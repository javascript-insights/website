⚙️ Console Settings Options:

  • Preserve Log: Keep logs between page refreshes
  • Selected Context Only: Show logs from current iframe only
  • Eager Evaluation: Preview expression results while typing
  • Autocomplete: Get suggestions while typing in console
  • Group Similar: Combine repeated messages
  • Log XMLHttpRequests: Show network requests in console

Tip: Access these settings via the console's gear icon ⚙️ or three-dot menu
