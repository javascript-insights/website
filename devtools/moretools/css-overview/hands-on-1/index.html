Low contrast text reduces readability, especially for users with vision impairments. DevTools can help identify and fix these issues.

Use DevTools to:

  • Find contrast issues using the CSS Overview panel, Issues tab, or Lighthouse report.
  • Fix issues with the Color Picker's suggested colors.
  • Emulate vision deficiencies to see how users experience your site.

Discover low contrast text

To find low contrast text:

  1. Open DevTools on your page.
  2. Use one of these panels to get a list of issues:
    • CSS Overview
    • Issues
    • Lighthouse

CSS Overview panel

To get an overview:

  1. Open CSS Overview and capture an overview.
  2. Check the Colors section for contrast issues and click an issue.

Issues tab

Note: This is a preview feature.

To enable and use:

  1. Enable contrast issues reporting in Settings > Experimental.
  2. Open the Issues tab and expand the contrast issues.

Lighthouse report

To run a report:

  1. Open Lighthouse in DevTools.
  2. Generate a report with Accessibility category selected.
  3. Check the Contrast section for issues.

Fix low contrast text

To fix an issue:

  1. Click a link to an affected element in the CSS Overview, Issues tab, or Lighthouse report.
  2. Open the Color Picker and use the suggested color to meet contrast guidelines.

Emulate vision deficiencies

To emulate vision deficiencies:

  1. Open DevTools.
  2. Go to the Rendering tab.
  3. Use the Emulate vision deficiencies dropdown to select a deficiency.
