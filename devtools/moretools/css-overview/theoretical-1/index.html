Welcome to the CSS Overview Tool Practice

Follow the instructions below to learn and practice using the CSS Overview tool in browser DevTools.

Instructions

  1. Open your browser's DevTools (usually by pressing F12 or Ctrl+Shift+I).
  2. Navigate to the "CSS Overview" tab.
  3. Click on "Capture Overview" to generate a report of the CSS used on this page.
  4. Review the report to identify any CSS issues or areas for improvement.

CSS Overview Game

Let's play a game to find and fix CSS issues!

Task: Use the CSS Overview tool to find the following issues on this page:

  • Identify any unused CSS rules.
  • Find any color contrast issues.
  • Check for redundant CSS properties.
  • Identify font size inconsistencies.
  • Find and resolve CSS specificity issues.
  • Detect and correct performance issues caused by CSS animations.

Once you've identified the issues, try to fix them by editing the CSS directly in the DevTools.

Hint: Look for the red highlighted text for potential issues!
