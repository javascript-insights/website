Security Tool Hands-On

Welcome to the Security Tool Hands-On website. This site will provide you with practical tasks to help you master the Security tool in browser DevTools.

Instructions

Open the browser DevTools (F12 or right-click and select "Inspect"). Navigate to the "Security" tab and complete the following tasks:

Task 1: Check HTTPS Status

Verify the HTTPS status of this page. Is it secure? If not, identify the issues.

Task 2: View Certificate Details

Inspect the SSL/TLS certificate details. Note down the issuer, validity period, and subject details.

Task 3: Identify Mixed Content

Find any mixed content on this page. List the insecure elements and their sources.

Task 4: Understand Security States

Review the security states of different elements on this page. What do the different states indicate?

Task 5: Implement Recommendations

Based on the Security tool's recommendations, suggest improvements to enhance the security of this page.
