Performance Monitor Tool Demo

Welcome to the Performance Monitor Tool demo! Use the buttons below to simulate different performance trends. Open your browser's DevTools and navigate to the Performance Monitor tab to observe the changes.

Instructions:

  • Click on the buttons to simulate different trends.
  • Observe the changes in the Performance Monitor tab in DevTools.
  • Analyze how each trend affects the performance metrics.

Explanation:

  • Simulate High CPU Usage: A busy-wait loop for 1 second to simulate high CPU usage.
  • Simulate Large JS Heap Size: Creates a large array to consume memory.
  • Simulate Many DOM Nodes: Adds 1000 new div elements to the DOM.
  • Simulate Many JS Event Listeners: Adds 1000 click event listeners to the body.
  • Simulate Multiple Documents: Adds 10 iframes to simulate multiple documents.
  • Simulate Multiple Document Frames: Adds 10 iframes to simulate multiple document frames.
  • Simulate High Layouts/sec: Adds 1000 absolutely positioned div elements to trigger layout calculations.
  • Simulate High Style Recalc/sec: Adds 1000 div elements with different colors to trigger style recalculations.
