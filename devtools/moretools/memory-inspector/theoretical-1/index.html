Memory Inspector Tool Demo

Welcome to the Memory Inspector tool demo. Use the buttons below to allocate different types of memory and inspect them using the Memory Inspector in DevTools.

Instructions

1. Open the DevTools in your browser (F12 or right-click and select "Inspect").

2. Navigate to the "Memory" tab.

3. Click on the "Take snapshot" button to capture the current memory state.

4. Use the buttons above to allocate different types of memory.

5. Take another snapshot after each allocation to see the changes in memory usage.

6. Analyze the memory snapshots to understand how different types of memory are allocated and managed.
