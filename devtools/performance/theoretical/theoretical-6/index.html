DevTools provides numerous ways to analyze frames per second:

  • Use the Frames section to view how long a particular frame took.
  • Use the FPS meter for a real time estimate of FPS as the page runs. See View frames per second in real time with the FPS meter.

The Frames section can show four types of frames:

  • Idle frame (white). No changes.
  • Frame (green). Rendered as expected and in time.
  • Partially presented frame (yellow with a sparse wide dash-line pattern). Browser did its best to render at least some visual updates in time. For example, in case the work of the main thread of the renderer process (canvas animation) is late but the compositor thread (scrolling) is in time.
  • Dropped frame (red with a dense solid-line pattern). Browser can't render the frame in reasonable time.
