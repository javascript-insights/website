Instructions

In this task, you will learn how to use the browser DevTools to analyze and improve the performance of a simple animation.

Tasks:

  1. Open the browser DevTools by pressing F12 or Ctrl+Shift+I (Windows) / Cmd+Opt+I (Mac).
  2. Navigate to the Elements tab and inspect the <div id="ball"> element. Observe how the styles change when you adjust the animation speed.
  3. Switch to the Console tab and use console.log to log the current animation speed. For example, type console.log(document.getElementById('speed').value); and press Enter.
  4. Go to the Performance tab and start recording a performance profile. Adjust the animation speed and observe how it affects the performance metrics.
  5. Analyze the performance profile and identify any potential bottlenecks or areas for improvement.
  6. Experiment with different animation speeds and observe the impact on CPU usage and frame rate.
  7. Document your findings and suggest ways to optimize the animation for better performance.
