Instructions

In this task, you will learn how to use the browser's DevTools to analyze and improve the performance of web interactions.

  1. Open the browser DevTools (usually by pressing F12 or Ctrl+Shift+I).
  2. Navigate to the Performance tab.
  3. Click the Start profiling and reload page button (the circle icon) to begin recording.
  4. Click the testButton1 and observe the performance metrics.
  5. Click the testButton2 and observe the performance metrics.
  6. Analyze the recorded performance data to identify any bottlenecks or areas for improvement.
  7. Try to optimize the code in index.js to reduce the interaction delay.

Use the DevTools to inspect the network requests, JavaScript execution time, and rendering performance. Document your findings and optimizations.
