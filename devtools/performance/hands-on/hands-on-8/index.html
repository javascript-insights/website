Performance Tool Hands-On

Instructions

Welcome to the Performance Tool Hands-On exercise. Follow the steps below to complete the task:

  1. Open the Developer Tools in your browser (usually by pressing F12 or right-clicking on the page and selecting "Inspect").
  2. Navigate to the "Performance" tab.
  3. Click the "Start Worker Tasks" button to initiate the worker tasks.
  4. Click the "Record" button in the Performance tab to start recording the performance.
  5. Wait for the worker tasks to complete (you will see a message in the console).
  6. Click the "Stop" button in the Performance tab to stop recording.
  7. Analyze the recorded performance data to understand the impact of the worker tasks on the page's performance.

Try to identify any performance bottlenecks and think about how you might optimize the code to improve performance.
