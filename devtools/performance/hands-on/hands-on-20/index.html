Maximum call stack size exceeded

In this hands-on exercise, we will explore the "Maximum call stack size exceeded" error in JavaScript.

This error occurs when a function calls itself recursively without a proper base case, leading to an infinite loop of function calls.

To trigger this error, we will create a simple recursive function that does not have a base case.

Once the error is triggered, we will use the browser's DevTools Performance tool to inspect the call stack and understand what went wrong.
