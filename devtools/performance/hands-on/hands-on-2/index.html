Instructions

Welcome to the Performance Hands-On exercise. Follow the steps below to complete the tasks:

  1. Open the browser DevTools by pressing F12 or Ctrl+Shift+I (Windows) / Cmd+Option+I (Mac).
  2. Navigate to the Network tab.
  3. Reload the page and observe the network requests.
  4. Identify the largest file being loaded and note its size and loading time.
  5. Switch to the Performance tab and start a recording.
  6. Reload the page and stop the recording after the page has fully loaded.
  7. Analyze the performance recording to identify any bottlenecks.
  8. Make changes to the code in index.js to optimize the performance based on your analysis.
