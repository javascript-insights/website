Performance Hands-On 21 - Counter

JavaScript Insights DevTools Performance Hands-On Counter
Back

This example demonstrates a simple counter that increments every second. The speed of the counter can be adjusted using a slider.

Instructions

  1. Open Chrome DevTools (F12 or Ctrl+Shift+I)
  2. Switch to the Performance tab
  3. Click the record button (circle icon)
  4. Interact with the counter and slider
  5. Stop recording
  6. Analyze the frame timeline and performance metrics
  7. Look for frame drops and notice how different speed settings affect frames
