Performance Tool Hands-On

Instructions:

  1. Open the browser DevTools (F12 or right-click and select "Inspect").
  2. Go to the "Performance" tab.
  3. Click on the "Record" button to start recording the performance.
  4. Click the "Run CPU Intensive Task" button and observe the performance recording.
  5. Stop the recording by clicking the "Stop" button.
  6. Analyze the recorded performance data:
    • Identify the CPU usage spikes.
    • Examine the call stack to understand which functions are consuming the most CPU time.
    • Look for any potential performance bottlenecks.
  7. Repeat the process with the "Run Simple Task" button and compare the results.
  8. Document your findings and insights.

Tasks:

  1. Identify the difference in CPU usage between the CPU Intensive Task and the Simple Task.
  2. Explain why the CPU Intensive Task consumes more resources.
  3. Suggest optimizations for the CPU Intensive Task to improve performance.
  4. Explore other tabs in DevTools (e.g., Network, Memory) to gather more insights about the page's performance.
  5. Share your findings with the class and discuss potential improvements.
