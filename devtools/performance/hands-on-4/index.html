Performance Tool Hands-On

Instructions:

  1. Open the browser DevTools (F12 or right-click and select "Inspect").
  2. Go to the "Performance" tab.
  3. Click "Record" and then run the CPU Intensive Task.
  4. Stop the recording and analyze the CPU usage.
  5. Repeat the process with the Simple Task and compare the results.
