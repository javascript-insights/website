Memory Leak Exercise

In this exercise, you will identify and fix a memory leak.

  1. Open the DevTools and go to the "Performance" tab.
  2. Click the "Create Memory Leak" button below multiple times.
  3. Take a heap snapshot and analyze the memory usage.
  4. Identify the memory leak in the code and fix it.
  5. Verify the fix by taking another heap snapshot and comparing the results.

Optimization Exercise

In this exercise, you will optimize the DOM manipulation code.

  1. Open the DevTools and go to the "Performance" tab.
  2. Click the "Manipulate DOM" button below and record the performance.
  3. Analyze the performance recording to identify bottlenecks.
  4. Optimize the code to reduce memory usage and improve performance.
  5. Verify the optimization by recording the performance again and comparing the results.
