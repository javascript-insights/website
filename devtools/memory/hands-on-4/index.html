How to Use the Memory Inspector

    Setup:

  1. Open Chrome DevTools by pressing F12 or right-clicking anywhere and selecting "Inspect"
  2. Click on the "Memory" tab in DevTools (if not visible, click >> to show more tabs)

    3. Taking Snapshots:

  3. Click "Take heap snapshot" in the Memory panel
  4. Wait for the initial snapshot to complete - this is your baseline

    5. Testing Memory Allocation:

  5. Click any of the buttons above to allocate different types of memory
  6. Take another heap snapshot after each allocation
  7. Compare snapshots by selecting them from the left panel to see memory changes

    8. Analysis:

  8. Look for your allocated objects in the snapshot comparison
  9. Observe the size and count of allocated objects
  10. Note how different data structures consume memory differently
