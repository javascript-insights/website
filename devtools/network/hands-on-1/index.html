To complete this hands-on exercise, please follow these steps:

Using Overrides:
  1. Locate the legs.html file in the file navigator in the Sources tool
  2. Opps, it is not there
  3. You will need to override it using the Network tool
  4. Right-click the legs.html file and select Override content.
  5. When prompted, enable overrides and select a folder for local modifications.
  6. Fix the horse's legs in the overridden file by adjusting the ASCII art.
  7. Save the changes and refresh the page to see the updated horse.
  8. To disable overrides, right-click the legs.html file and select Disable overrides.
  9. To remove overrides, right-click the legs.html file and select Remove overrides.
